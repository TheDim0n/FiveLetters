# FiveLetters
Игра "5 букв"

## Platforms
Поддерживаемые платформы:
- Windows (already in releases)
- Linux (manual build)
- MacOS (manual build)

## Stack
- `Rust` programming language
- `Tauri`
- `React` (with `Typescript`)

## Run development
```
cargo tauri dev
```

## Run build
```
cargo tauri build
```
